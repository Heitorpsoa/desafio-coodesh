# desafio-coodesh

## Instalação

#### Requisitos
    npm
    node acima da versão 20

Entre na root do diretório, vá na pasta coodesh-back e execute
```bash
  npm start
```

Em outro terminal execute o front end

```bash
  flutter run
```

## Bugs
